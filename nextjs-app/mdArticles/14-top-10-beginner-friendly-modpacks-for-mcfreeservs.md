## Top 10 beginner‑friendly modpacks that run great on HostChicken

If you’re new to modpacks, start with something friendly and stable. These picks focus on smooth performance, clear goals, and easy onboarding—perfect for playing with friends. Always double‑check versions and read each pack’s notes.

Note: Pack availability shifts over time. Treat this as a style guide—look for similar packs with these traits if names change.

### 1) A modern Fabric starter

- Loader: Fabric (1.20.x)
- Why it’s great: Uses Lithium/FerriteCore/Starlight for speed, adds a few exploration and building mods without overwhelming you.
- Server notes: View distance 8–10, simulation 5–6. Easy to host with low RAM.

### 2) Lightweight Create experience

- Loader: Forge
- Why it’s great: The Create mod adds clever mechanical contraptions without heavy tech trees. Great for creativity.
- Server notes: Keep automation moderate; avoid entity spam.

### 3) Adventure and exploration lite

- Loader: Fabric
- Why it’s great: New biomes, structures, and small dungeons that feel like “vanilla++.”
- Server notes: Pre‑generate chunks around spawn to smooth exploration.

### 4) Cozy building pack

- Loader: Fabric or Forge
- Why it’s great: Adds decorative blocks, furniture, and tools for towns and bases.
- Server notes: Perfect for SMPs focused on community builds.

### 5) Tech sampler

- Loader: Forge
- Why it’s great: Light tech progression—enough machines to learn automation without a giant manual.
- Server notes: Limit chunk loaders; space out machines.

### 6) Magic lite

- Loader: Forge
- Why it’s great: Gentle magic with spells or rituals that don’t flood you with systems.
- Server notes: Keep entity farms under control.

### 7) School‑friendly learning pack

- Loader: Fabric
- Why it’s great: Adds maps, recipe viewers, and creative tools—great for teaching and collaborative builds.
- Server notes: Enable backups; encourage student teams.

### 8) Performance‑boosted vanilla+

- Loader: Fabric
- Why it’s great: Feels like vanilla but faster and prettier with optional client‑only mods.
- Server notes: Minimal server overhead; very stable.

### 9) Exploration with quests

- Loader: Forge
- Why it’s great: Gentle quest lines guide new players through mods without pressure.
- Server notes: Allocate moderate RAM; keep simulation distance ~6.

### 10) Friends and family SMP pack

- Loader: Fabric
- Why it’s great: Balanced set of QoL mods, easy server setup, and safe defaults.
- Server notes: Perfect for whitelisted groups and community rules.

### How to choose for your group

- Start small: Fewer mods means fewer crashes and easier updates.
- Pick a theme: Building, exploration, light tech—focus helps.
- Check requirements: If your friends have mixed PCs, fabric‑based light packs are forgiving.

### Server setup tips

- Enable daily backups.
- Keep view distance 8–10 and simulation 5–6.
- Avoid always‑on redstone clocks and huge hopper fields.

### Make it welcoming

- Publish simple rules: be kind, no griefing, teamwork first.
- Use a custom domain so joining is easy.
- Help newcomers with a one‑page “how to install” guide.

With the right beginner‑friendly pack, HostChicken feels smooth and inviting—so everyone can explore, build, and laugh together without fighting settings.
